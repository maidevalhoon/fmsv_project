# LLM-Guided SAT-Based ATPG 

This project implements an LLM-guided SAT-based Automatic Test Pattern Generation (ATPG) framework, where an LLM acts as a semantic heuristic oracle to guide a SAT solver.

## File Structure
```
.
├── papers/                 # PDFs, problem statements, and guidelines
├── venv/                   # Python virtual environment (ignored by git)
├── circuit.v               # Original test circuit (RTL)
├── circuit_netlist.v       # Gate-level netlist generated by Yosys
├── circuit.json            # Structured JSON netlist generated by Yosys
├── netlist_parser.py       # Python script to parse JSON into structural nodes
├── synth.ys                # Yosys synthesis script
├── requirements.txt        # Python dependencies
├── progress_report.md      # Team tracking file
├── .gitignore              # Git ignore rules
└── README.md               # Setup and project info
```

## Setup Instructions

### 1. macOS
1. **Install Yosys**: `brew install yosys`
2. **Clone & CD**: `git clone <repo> && cd <repo>`
3. **Synthesis**: `yosys synth.ys` (Generates `circuit_netlist.v`)
4. **Python Env**: 
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt # (To be created)
   ```

### 2. Linux (Ubuntu/Debian)
1. **Install Yosys**: `sudo apt update && sudo apt install yosys`
2. **Clone & CD**: `git clone <repo> && cd <repo>`
3. **Synthesis**: `yosys synth.ys`
4. **Python Env**: 
   ```bash
   sudo apt install python3-venv
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt # (To be created)
   ```
